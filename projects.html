AngleSharp.Core

HTML, CSS parser. Required helpers and parsing for mime types, urls, source sets, and many more. Provides extension points, which are then used by the other projects.

Currently in beta. Considered stable.

AngleSharp.Scripting

Scripting solutions for integrating a JavaScript engine (Jint) and advanced C# parsing (DLR, scriptcs or Roslyn). Provides DOM bindings (and binding helpers).

Currently in alpha stage. Leaving experimental stage soon.
