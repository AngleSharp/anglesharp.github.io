AngleSharp.Core

HTML, CSS parser. Required helpers and parsing for mime types, urls, source sets, and many more. Provides extension points, which are then used by the other projects.

Currently in beta. Considered stable.

AngleSharp.Css

A full CSS parser and CSSOM provider to include support for stylesheets, inline styles, and style computations within AngleSharp.

Currently prior to publish. Available once v0.10 of AngleSharp.Core hits the road.

AngleSharp.Io

IO modules and helpers, including - but not limited to - improved (and more) requesters, storage abilities, and caching mechanisms.

Currently in alpha stage. Leaving experimental stage soon.
AngleSharp/AngleSharp.Io

AngleSharp.Scripting

Scripting solutions for integrating a JavaScript engine (Jint) and advanced C# parsing (DLR, scriptcs or Roslyn). Provides DOM bindings (and binding helpers).

Currently in alpha stage. Still in experimental stage.
